# Quick Plan: Add GitHub Action for Cloudflare Pages Deployment

## Task 1: Create GitHub Actions workflow for Cloudflare Pages deploy

**Files:** `.github/workflows/deploy.yml` (new)
**Action:** Create a GitHub Actions workflow that:
1. Triggers on push to `main` branch
2. Installs Node.js and dependencies
3. Builds with `NITRO_PRESET=cloudflare_pages`
4. Deploys to Cloudflare Pages using `cloudflare/wrangler-action`
5. Passes all required environment variables from GitHub Secrets

**Environment variables needed (from .env.example):**
- `CLOUDFLARE_API_TOKEN` — Cloudflare API token with Pages edit permission
- `CLOUDFLARE_ACCOUNT_ID` — Cloudflare account ID
- `SUPABASE_URL`, `SUPABASE_KEY`, `SUPABASE_SERVICE_ROLE_KEY`
- `NUXT_PUBLIC_APP_URL`
- `NUXT_QR_TOKEN_SECRET`
- All `NUXT_APPLE_*`, `NUXT_GOOGLE_*`, `NUXT_SAMSUNG_*` wallet secrets

**Verify:** YAML is valid, references correct scripts and secret names
**Done:** Workflow file exists at `.github/workflows/deploy.yml`
